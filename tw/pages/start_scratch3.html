使用 Scratch 3.0 作為操作端控制介面

Scratch 3.0 Beta For Zenbo

從現在起，您可以選用 Scratch 3.0 作為您操控 Zenbo 的程式！ 前往

在開始之前，請先閱讀以下注意事項：

1. Scratch 3.0 目前官方版本為測試(beta)版本，連結為：https://beta.scratch.mit.edu/，但是官方測試版本目前無法載入 Zenbo Scratch 積木，因此 Zenbo Scratch 團隊修改了一個已經包含了 Zenbo Scratch 積木的網站，提供大家試試！

2. 從現在起，您可以使用電腦，智慧型手機，或平板電腦連到這個網站使用！

3. Zenbo Scratch 團隊無法保證使用的可靠度，僅為提供 Scratch 3.0 的試用，若您對網站可靠度有需求，建議先使用目前版本的 ScratchX 網站。

4. 若您已了解上述內容，再閱讀下面使用說明後，請點選"前往"按鈕！ 之後按照下面的說明讀取 Zenbo Scratch 的積木：

進入 Scratch 3.0 網頁後，請按照下面步驟讀取 Zenbo Scratch 積木：

1. 請點選左下角更多方塊的按鈕，選擇其中一個 Zenbo Scratch 積木（ 專業版 / 教育版 )：

2. 就可以看到 Zenbo Scratch 積木被新增了！

準備好了嗎？ 出發進入 Scratch 3.0 網頁囉！
