Scratch Blocks

The followings are all Zenbo Scratch block instructions：

1. Set the variable of Zenbo’s IP and you could get the Zenbo’s IP in Zenbo Scratch app.

2. Control Zenbo with IP (INPUT IP) move (FORWARD / BACKWARD) ( 0.25 / 0.5 / 0.75 / 1.0 / 1.25 / 1. 5/ 1.75 / 2.0) meter at the speed (SLOW / NORMAL / FAST).

3. Control Zenbo with IP (INPUT IP) to stop moving..

4. Control Zenbo’s head at IP (INPUT IP) to turn ( LEFT / RIGHT/ UP / DOWN) ( 0 / 15 / 30 / 45) degrees. The degree is absolute angle. If the Zenbo is up 15 degree and you choose the head up 15 degree block, Zenbo would not move.

5. Control Zenbo’s body with IP (INPUT IP) to turn (left/ right) (0/ 15/ 30/ 45/ 60/ 75/ 90/ 105/ 120/ 135/ 150/ 165/ 180)degrees.

6. Control Zenbo with IP (INPUT IP) to (STOP / MOVE FORWARD / TURN LEFT/ TURN RIGHT) continously.

7. Control Zenbo with IP (INPUT IP) to do built-in movement (#). About the movement corresponding to #, please reference the official document: [click here]. See “Built-in movement“ section..

8. Control Zenbo with IP (INPUT IP) to stop built-in movement.

9. Control Zenbo with IP (INPUT IP) to set expression(#). About the expression corresponding to #, please reference the official document: [click here]. See “Facial Expression“ section.

10. Control Zenbo with IP (INPUT IP) to say (Hi, 你好 / 看這裡/ WoW / YA).

11. Control Zenbo with IP (INPUT IP) to say (INPUT STRING).

12. Control Zenbo with IP (INPUT IP) of stream (TTS/ Music/ Alarm/ Notification).

13. Control Zenbo with IP (INPUT IP) to speak (INPUT STRING) using speak speed (L1/ L2/ L3/ L4/ L5/ L6/ L7/ L8). The L1 correspond to the slowest speech speed, L2 to L8 is in ascending order, and the L8 is the fastest speech speed.

14. Register the Dialog System to listen specific context (1/ 2/ 3/ 4/ 5) at Zenbo with IP (INPUT IP). Usually use together with the block “15. start listen context”.

15. Control Zenbo with IP (INPUT IP) to start listening.

16. Do something when Zenbo at IP (INPUT IP) listened context (1/ 2/ 3/ 4/ 5). Use together with the block “14. register the dialog system to listen specific context”.

17. Delete all registered context of dialog system at Zenbo with IP (INPUT IP).
